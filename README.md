### React-Redux JWT Authentication Boilerplate

# Client
### Installation
```
yarn install
```

### Development
```
yarn start
```

### Production

First build the project, then run the server.
```
yarn build
```

# Server

### Run
with node
```
node server/
```

with nodemon
```
nodemon server/
```

with PM2
```
pm2 start server/
pm2 monit
```
